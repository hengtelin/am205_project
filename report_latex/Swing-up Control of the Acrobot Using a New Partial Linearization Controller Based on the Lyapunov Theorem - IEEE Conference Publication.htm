Swing-up Control of the Acrobot Using a New Partial Linearization Controller Based on the Lyapunov Theorem

Abstract:
In this paper, a design of swing-up controller of an Acrobot is proposed. The Acrobot is a model of a horizontal bar gymnast, it has only one actuator at the joint of both links and is a typical example of a nonlinear underactuated system. Since the robot has a large range of motion, the Acrobot has a highly nonlinear characteristic and swing-up control of the Acrobot is a difficult problem. The Acrobot has a simple structure and its control objective is clear, hence it is used as a controlled object of not only the nonlinear underactuated control problem but also a remote network control problem recently. For swing-up control of the Acrobot, this paper proposes a controller, which uses pumping energy of the first link. In the controller, a newly proposed partial linearization is used. The nonlinear feedback is constructed based on the Lyapunov theorem. The new feature of the proposed partial linearization method is to guarantee that the linearized sub-systems track the time-variant reference function, which is not guaranteed in previous methods. Numerical simulation is performed to show the effectiveness of the proposed method

SECTION I.

Introduction

An underactuated system means a system that controls all degree of freedom using less actuators in number than the number of generalized coordinates. Thus, the design method of controllers of underactuated systems gives some reductions of numbers of necessary actuators, of the cost and of the weight of systems, hence this problem is interesting research topics in industry. Moreover, the movement of underactuated systems is considered to simulate the action of living creatures, and the research of control of underactuated system is indispensable for the realization of the robots that imitate the movement of man and animal.

The Acrobot[1] which is intended to model a horizontal bar gymnast by a robot is one of the typical examples of nonlinear underactuated systems and is a two degree of freedom robot that is constructed by two links and only one actuator at the joint of both links. The first joint of the Acrobot that did not have actuator is a model of human hand gripping the horizontal bar, and the actuated joint of two links is corresponding to the human hip.

Recently, the Acrobot is used as a controlled object of not only the nonlinear underactuated control problem but also a remote network control problem because the Acrobot is a simple structure and an objective of its control is clear.

Many controllers have been proposed for the Acrobot, such as energy based controllers[1][2], controllers based on partiallinearization[1][3], tracking controller[4], back stepping controller[5], a controller based on the motion of the real gymnast[6] etc.. In particular, Spong first considered control problem of the Acrobot[1]. In the paper [1], he derived a mathematical model of the Acrobot and proposed two swing-up controllers via using partial linearization method.

Partial linearization[8][9][10] is one of the nonlinear control method based on structure theories of systems and it divides systems into linearized subsystems and nonlinear subsystems using a change of variables and a nonlinear feed-back. The linearized subsystems can track a fixed reference value by a PD controller which is included in the nonlinear feedback.

The first controller in [1] consists of partially linearized terms of the first link. The reference angle of first link is set to to stabilize at the upright position, and the second link which is nonlinear subsystem, so-called Zerodynamics, was set into rotating mode by tuning control parameters.

The second one is a partially linearized term of the second link. The reference angle of second link is decided so that the energy of the first link may increase, and with making the second link follow the angle using partial linearization, the first link is swung up.

However, the second controller in [1] has two problems, the first one is that it does not guarantee for the second link to reach the up-right position simultaneously when the first link reaches up-right position. The second problem is the second controller is not to guarantee the second link can track the reference angle by the PD controller in the nonlinear feedback, because the reference angle used in it is a time-variant function. For the first problem, a new function which is designed such that the second link may reach up-right position simultaneously when the first link reaches up-right position is proposed by our group in [3].

In this paper, to solve the second problem a new swig-up controller is proposed to modify the second controller of [1] and controller of [3] for the time-variant reference angle in the second controller.

The function proposed by our group in [3]is used as the reference angle of the second link in the proposed controller. The function is designed such that the second link may reach up-right position simultaneously when the first link reaches up-right position.

A new tracking controller of the second link is given. The controller is based on a new partial linearization method using a nonlinear feedback with Lyapunov theorem and it guarantees that the linearized sub-systems track the reference time-variant function.

In Section II, dynamics of the Acrobot depicted in Fig. 1 is obtained.

In Section III, a new swing-up controller of the Acrobot modified previous controller in [1][3] is proposed. The proposed controller uses pumping energy of the first link by making the angle of the second link track to the reference angle using the partial linearization method. The reference angle of the second link is designed to make the two links reach to up-right position simultaneously[3].

In Section IV, in order to show the effectiveness of the proposed methods, simulation results of comparing the proposed method with the previous one are shown.

SECTION III.

A Swing up Controller of the Acrobot

This section proposes a new swing-up controller by extending the result in [1]. In [1], since deciding the reference angle of the second link so that the energy of the first link may increase, and the first link is swung up by making the second link follow to it using partial linearization.

However, the reference angle used in previous controller[1] does not guarantee that both links are swung up to up-right position simultaneously, and it is not to guarantee the second link can track the reference angle by the PD controller in the nonlinear feedback because the time-variant function is used as reference angle.

Then, for function of the reference angle, the function, which is designed in [3] such that the second link may reach up-right position simultaneously when the first link reaches up-right position is used.

For the tracking control of the second link, the new partial linearization method using a new nonlinear feedback based on Lyapunov theorem is proposed to guarantees that the linearized sub-systems track the time-variant function.

In order to show the effectiveness of proposed method, the numerical simulations of swing-up control of the Acrobot are performed. SECTION V.

Conclusion

In this paper, the new swing-up controller of the Acrobot is proposed. The proposed controller make the angle of the second link track to the reference angle, which is increasing energy of the first link, by using the new partial linearization method.

For the reference angle of the second link, the function which is designed so that two link can be swung up simultaneously[3] is used, it is not guaranteed in previous method.

For the tracking control of the second link, the new partial linearization method, which guarantees that the linearized sub-systems track the reference time-variant function, is proposed. The proposed partial linearization method is considered with dynamics of the reference function of the second link by using a nonlinear feedback that is constructed based on Lyapunov theorem.

In numerical simulation of comparing the proposed method with the previous one, the angle of the second link using proposed method can be track time-variant function of the reference angle, and the Acrobot can be swung up earlier than previous method.
